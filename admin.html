👥 Thêm Tài Khoản Admin Mới

Tài khoản admin được quản lý qua Supabase Authentication. Để thêm admin mới, làm theo các bước sau:

1️⃣
Mở Supabase Dashboard
Vào supabase.com/dashboard → chọn project Bộ Tộc
2️⃣
Vào Authentication → Users
Bấm nút 「Invite User」 → nhập email của admin mới → gửi lời mời
3️⃣
Admin nhận email → đặt mật khẩu
Sau khi nhận email mời, admin mới bấm link → đặt mật khẩu → đăng nhập bình thường vào trang admin.html
⚠️
Xóa tài khoản admin
Vào Authentication → Users trong Supabase → tìm user → bấm Delete
